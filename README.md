
# Customer Segmentation using Clustering (K-Means & Hierarchical)

---

## 📖 Overview

This project demonstrates the application of **unsupervised machine learning techniques** to discover hidden patterns in data.

Using **K-Means** and **Hierarchical Clustering**, the project segments data into meaningful groups and provides insights that can support data-driven decision-making.

---

## 🎯 Objectives

* Apply clustering algorithms to real-world data
* Compare K-Means and Hierarchical clustering methods
* Visualize clusters and interpret patterns
* Extract actionable insights from grouped data

---

## 🧠 Algorithms Used

* **K-Means Clustering**
* **Hierarchical Clustering (Agglomerative)**

---

## ⚙️ Technologies & Tools

* Python 🐍
* NumPy
* Pandas
* Matplotlib / Seaborn
* Scikit-learn

---

## 📊 Project Workflow

1. Data Loading & Exploration
2. Data Preprocessing
3. Feature Scaling
4. Applying K-Means Clustering
5. Finding Optimal Clusters (Elbow Method)
6. Hierarchical Clustering (Dendrogram)
7. Visualization of Clusters
8. Insights & Interpretation
